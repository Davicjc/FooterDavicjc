Demo do Footer

Esta é uma página de demonstração para visualizar o footer.

Como funciona?

O footer aparece automaticamente no canto inferior direito da página.

Você pode arrastar o footer para qualquer posição clicando e segurando.

Características

  • Discreto - apenas texto branco
  • Posição fixa no canto inferior direito
  • Arrastável para qualquer lugar
  • Responsivo para mobile

Instalação

Basta adicionar o script ao seu HTML:

<script src="footer-davicjc.js"></script>
